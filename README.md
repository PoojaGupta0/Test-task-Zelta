### Installation

1. To Unzip this project:

   ```bash
   unzip playwright_plus-master.zip
   ```

2. Goto the project direcatory:

   ```bash
    cd playwright_plus-master
   ```

3. For run test cases:

   ```
   cd playwright_plus
   run 'python test_cases.py'
   ```
